# Task Tracker - Проектная практика

Fullstack-приложение для управления задачами с использованием современного стека технологий.

## Описание проекта

Task Tracker - это веб-приложение для управления проектами и задачами, разработанное в рамках проектной практики. Проект включает в себя:

- **Backend API** на базе FastAPI с модульной архитектурой
- **Frontend** на React с TypeScript и Feature-Sliced Design
- **DevOps инфраструктуру** с мониторингом и логированием

### Основные возможности

- Управление задачами с различными статусами
- Система тегов для категоризации задач
- Чек-листы для детализации задач
- Drag & Drop интерфейс для удобного управления
- Мониторинг производительности через Prometheus и Grafana
- Централизованное логирование через Loki

## Архитектура

### Backend

Модульный монолит с разделением на слои:
- API endpoints (FastAPI роутеры)
- Service layer (бизнес-логика)
- Repository layer (работа с БД)
- Models (SQLAlchemy модели)
- Schemas (Pydantic DTO)

### Frontend

Feature-Sliced Design архитектура:
- `app/` - конфигурация приложения
- `pages/` - страницы приложения
- `features/` - бизнес-функциональность
- `entities/` - бизнес-сущности
- `shared/` - переиспользуемый код

### Infrastructure

- Контейнеризация через Docker
- Reverse proxy на Nginx
- Мониторинг: Prometheus + Grafana
- Логирование: Loki + Promtail
- CI/CD через GitHub Actions
